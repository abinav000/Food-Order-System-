# Food App - Spring Boot Application

A comprehensive food management system built with Spring Boot.

## Features
- Product management
- Supplier management
- Purchase order management
- Customer returns handling
- Inventory tracking

## Technologies Used
- Java 17
- Spring Boot 3.4.12
- Spring Data JPA
- MySQL
- Maven

## Getting Started

### Prerequisites
- Java 17 or higher
- Maven 3.6+
- MySQL database

### Running the Application
1. Clone the repository
2. Configure database connection in `application.properties`
3. Run `mvn spring-boot:run` from the springapp directory

## Project Structure
- `controller/` - REST API controllers
- `model/` - Entity classes
- `repository/` - Data access layer
- `service/` - Business logic layer
